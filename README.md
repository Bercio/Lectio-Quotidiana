# Lectio Quotidiana

Lectio Quotidiana è un'applicazione Android che rende disponibili ogni giorno i testi per
la meditazione quotidiana secondo il 
[calendario biblico](https://www.famigliedellavisitazione.it/wp/wp-content/uploads/Lectio/Calendari/2026-Calendario%20biblico%20Piccola%20famiglia%20dell%27annunziata.pdf)
della Piccola Famiglia dell'Annunziata.

L'app permette di consultare la lettura del giorno (presa dalla traduzione CEI 2008
disponibile su [www.bibbiaedu.it](www.bibbiaedu.it) ) direttamente dal proprio dispositivo
e offre la possibilità di scaricarle in anticipo per la consultazione offline, garantendo
la disponibilità dei testi anche in assenza di connessione internet.

## Requisiti di sviluppo

*   Android Studio Hedgehog (2023.1.1) o superiore.
*   Android SDK 34.
*   Gradle 8.7.
